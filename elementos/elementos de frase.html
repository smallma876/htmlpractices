Concepto de elementos de frase

Son elementos que ayudan a dar formato al texto del documento sin css por ejemplo

Ejemplo de conceptos de elementos de frase

Escribimos un parrafo cualquiera y lo encerramos con la etiqueta strong, de esta manera la palabra encerrada se sombreara de negro.
Otro ejemplo de elementos de frase es la etique em y sirve para poner en diagonal la palabra seleccionada.
Otro ejemplo tambien es la etiqueta small y sirve para achicar la frase dentro de la etiqueta, se usa para poner pies de pagina por ejemplo o citas de pag.
Tambien para resaltar se usa el mark para resaltar la palabra.
Tambien se usa para elevar numero con potencias por ejemplo con la etiqueta sup y lograr los siguientes ejemplos 203 H2O
